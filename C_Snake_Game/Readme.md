# Console Snake Game

Welcome to the Console Snake Game, a simple and nostalgic implementation of the classic Snake game that was popular on old button mobile phones. This game is written in C and utilizes the `conio.h` library to provide a console-based interface.

## Features

- **Classic Gameplay:** Relive the timeless fun of guiding the snake to eat food and grow longer.
- **Retro Console Interface:** Enjoy the simplicity and charm of playing in a console environment.
- **Responsive Controls:** Use arrow keys (a, s, w, d) for precise control over the snake's movement.
- **Scoring System:** Earn points by consuming fruits, and the score is displayed at the end of each game.
- **Dynamic Tail:** Watch the snake's tail grow as it consumes more food.
